返回码规定          客户端   <=======> 服务端

---------------------------------------------------------------通用code
404=不存在的请求code
500=服务端执行异常
200=请求成功
-------------------------------------------------------------------登录
0001=用户名或密码错误
0002=登录成功
-------------------------------------------------------------------注册
0003=用户名已存在
0004=注册成功
0005=数据库保存注册数据失败
0006=邮箱地址已存在
------------------------------------------------------------删除|添加好友
0007=添加好友成功
0008=数据库保存添加好友数据失败
0009=数据库逻辑删除好友成功
0010=数据库逻辑删除好友失败
0011=该用户已是好友

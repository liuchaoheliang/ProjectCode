常见问题
1、什么是重庆精品预售活动？
本次活动是重庆农村商业银行与手机银行联盟网站合作举办的精品预售活动，采取线上预售、线下各支行网点提货的方式进行商品销售，客户在活动网站预订本地精品商品后，于提货点自行提取，便可以享受劲爆优惠价格够买活动商品。
2、活动的网址是？
本次活动的网址是“cq.o2obill.com”。
3、如何参加重庆精品预售活动？
客户只需注册、登陆本次活动网址（cq.o2obill.com），即可参加本次精品预售活动。
4、活动的时间是多久？
预售期：2014年5月10日-5月31日（该时间客户可以登陆活动页面购买商品） 提货期：2014年6月10日-6月30日（该时间客户可以去预订的支行网点提货）
5、活动商品购买流程是怎样的？

（1）预售期内，客户注册并登陆活动页面（cq.o2obill.com），挑选商品进行预定；

（2）客户选择所在区域、商圈、自提网点，填写姓名，确认并进行支付；

（3）用户支付成功后，系统会进行记录，待预售期截至后一并向用户发送验证码（验证码按订单发送，一笔订单一个码）；

（4）提货期内，客户至预定时选择的支行网点进行提货，出示验证码短信并报出姓名，即可完成提货。

6、预售期、配货期、提货期分别是多久？

预售期：2014年5月10日-5月31日

配货期：2014年6月1日-6月9日

提货期：2014年6月10日-6月30日

7、商品可以退换货吗？

1）本次活动支持7天无理由退换货。欧贝莉橄榄油、百加得冰锐朗姆预调酒，如提货商品外包装完整，没有使用过，不影响二次销售，可支持退货或者换货。奶牛梦工厂牛奶卡一经提货，概不退换货。

2）预售期内，用户可以取消订单或修改订单信息，重庆精品预售平台会在五个工作日为用户退款。预售期结束之后，恕不能取消订单。

3）如客户在支付款项时使用了积分，则在退款时会优先将积分退到用户精品预售平台账户里。

4）提货时敬请察看外包装是否完整，有破损的请拒绝收货并要求更换，请保留外包装7天（数码产品请保留15天）以便有质量问题进行退换货。

5）人为损坏的货品恕不能退换。

8、有哪几个网点参与本活动？
重庆主城区内70家已放置ipad终端的网点，此处请行方添加网点名单（包含所在区域、商圈、地址、电话）
9、支付方式有哪些？
客户除了可以使用历史活动送出的积分外，还可以使用现金支付方式参与本次活动，现金支付方式包括重庆农商行江渝手机银行卡及支付宝付款方式。
